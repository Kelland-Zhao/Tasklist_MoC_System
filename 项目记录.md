## 2026-06-02 保养/点检修改页 Toast 规范化

`Tasklist_MoC_Modify_JS.html`（保养修改，15 处 Swal）+ `Inspection_Edit_JS.html`（点检修改，11 处 Swal）全部对齐 §8：
- 两文件开头新增双语 helper `swalTitle`/`swalHtml`。
- 加载：`icon:info` 静态框 → `didOpen:()=>swal.showLoading()` 旋转态（页面加载 + 提交等待）。
- 成功：`加载成功`/`操作成功` → `timer:1500 + showConfirmButton:false`。
- 错误（加载失败/操作失败/系统错误/输入错误）：补 `confirmButtonText:'确定 / OK'`，详情用 `swalHtml` 双语。
- 无法编辑（warning 提示）：`text` → `swalHtml` 双语 + `确定 / OK`。
- 确认提交：`icon:question` → `warning` + `confirmButtonColor:'#E60012'` + 双语按钮（确认提交 / Submit、取消 / Cancel）。
- 四个 input 录入框（变更原因/编辑/新增/删除条数）：补双语按钮（下一步 / Next、取消 / Cancel）。
- Inspection_Edit 沿用其 `swal`（小写，Sweetalert2_js 同时挂 Swal/swal）。
- `node --check` 两文件语法均 OK。GAS 页含 google.script.run，未本地预览。
